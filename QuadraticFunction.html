What is a Quadratic Function?

Definition: A quadratic function is a polynomial function of degree 2. It has the general form:
f(x) = ax¬≤ + bx + c
where:
  • a, b, c are constants (a ‚â† 0)
  • a is the coefficient of x¬≤ (leading coefficient)
  • b is the coefficient of x (linear coefficient)
  • c is the constant term
  • x is the variable

Key Properties:

  • Degree: The highest power of x is 2
  • Graph: The graph is a U-shaped curve called a parabola
  • Vertex: The highest or lowest point on the parabola
  • Axis of Symmetry: A vertical line through the vertex
  • Direction: If a > 0, parabola opens upward; if a < 0, opens downward
Important: The coefficient 'a' must be non-zero. If a = 0, the function becomes linear (bx + c), not quadratic.

Examples of Quadratic Functions:

  • f(x) = x¬≤ + 5x + 6 (a=1, b=5, c=6)
  • f(x) = 2x¬≤ - 3x + 1 (a=2, b=-3, c=1)
  • f(x) = -x¬≤ + 4 (a=-1, b=0, c=4)
  • f(x) = 3x¬≤ (a=3, b=0, c=0)

Factoring Trinomials: ax¬≤ + bx + c

Definition: Factoring a trinomial means expressing it as a product of two binomials.

This process is the reverse of multiplying binomials.

Why Factor Quadratics?

  • To solve quadratic equations easily
  • To find zeros/roots of the function
  • To simplify algebraic expressions
  • To understand the behavior of quadratic functions

The Zero Product Property

Zero Product Property: If the product of two or more factors equals zero, then at least one of the factors must be zero.
If AB = 0, then A = 0 or B = 0 (or both)

Mathematical Expression:

If (x - a)(x - b) = 0, then x - a = 0 or x - b = 0

Therefore: x = a or x = b

Why is This Property Important?

  • It allows us to solve quadratic equations by factoring
  • Once we factor a quadratic, we can set each factor equal to zero
  • This gives us the solutions (roots) of the equation
  • The solutions are also called zeros, roots, or x-intercepts

How to Use the Zero Product Property:

  1. Factor the quadratic equation completely
  2. Set the equation equal to zero
  3. Set each factor equal to zero
  4. Solve each resulting equation
  5. Check your solutions
Important: The Zero Product Property only works when one side of the equation equals zero. If the equation is not equal to zero, you must first rearrange it.

Visual Understanding:

The solutions to a quadratic equation are the x-values where the parabola crosses the x-axis (where y = 0). These are also called the roots or zeros of the function.

Completing the Square Method

  4. Factor the left side as a perfect square
  5. Apply the square root property

Summary & Key Takeaways

Methods for Solving Quadratic Equations:

1. Factoring Method

  • Best for: Equations that factor easily
  • Steps: Factor ‚Üí Apply Zero Product Property ‚Üí Solve
  • Limitation: Not all quadratics factor nicely

2. Completing the Square

  • Best for: Any quadratic, especially when deriving formulas
  • Key: Add (b/2)¬≤ to create a perfect square trinomial
  • Result: (x + h)¬≤ = k format, then use square root property

3. Quadratic Formula

  • Best for: ANY quadratic equation (most versatile)
  • Formula: x = [-b ¬± ‚àö(b¬≤ - 4ac)] / (2a)
  • Advantage: Always works, even when factoring is difficult

Which Method to Use?

  • Quick check: Try factoring first (if it's obvious)
  • When a = 1 and b is even: Completing the square is efficient
  • When in doubt: Use the quadratic formula (it always works!)
  • Check the discriminant first: Know what to expect before solving
Remember: All three methods will give you the same answer‚Äîchoose the one that works best for the problem at hand!
