What is a Quadratic Equation?

A quadratic equation is a polynomial equation of the second degree. It's called "quadratic" because the highest power of the variable is 2 (from the Latin word "quadratus" meaning square).

ax¬≤ + bx + c = 0

Key Components:

a = coefficient of x¬≤ (cannot be zero!)

b = coefficient of x

c = constant term

x = the variable we're solving for

Examples of Quadratic Equations:

x¬≤ + 5x + 6 = 0

a = 1, b = 5, c = 6

2x¬≤ - 8x + 3 = 0

a = 2, b = -8, c = 3

x¬≤ - 16 = 0

a = 1, b = 0, c = -16

Why Do We Need the Quadratic Formula?

While some quadratic equations can be solved by factoring, many cannot be easily factored. The quadratic formula provides a universal method that works for ANY quadratic equation, making it an essential tool in mathematics!

Important Note: The quadratic formula will give us the values of x where the parabola crosses the x-axis. These are called the roots or solutions of the equation.

What Does This Formula Do?

This formula gives us the solutions (values of x) for ANY quadratic equation in the form ax¬≤ + bx + c = 0. It's a universal tool that always works, no matter what values a, b, and c have!

Breaking Down Each Part:

The Numerator: -b ¬± ‚àö(b¬≤ - 4ac)

-b (negative b)

This is the opposite of your b coefficient. If b = 5, then -b = -5. If b = -3, then -b = 3.

¬± (plus-minus symbol)

This symbol means we'll get TWO answers: one using addition (+) and one using subtraction (-).

That's why quadratic equations can have up to 2 solutions!

‚àö(b¬≤ - 4ac) (the square root part)

This is twice the coefficient of x¬≤. We divide the entire numerator by this value.

Important: The entire expression -b ¬± ‚àö(b¬≤ - 4ac) must be divided by 2a, not just part of it!

How to Use the Formula - The Process:

1 Write equation in standard form

2 Identify a, b, and c

3 Substitute into the formula

4 Simplify carefully
