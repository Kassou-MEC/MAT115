Absolute Value Equations & Inequalities

Understanding and Solving from Basics to Advanced

What is Absolute Value?

The absolute value of a number is its distance from zero on the number line, regardless of direction.

Mathematically, for any real number x:

|x| = { x, if x ≥ 0; -x, if x < 0 }

Distance Interpretation

|x| represents distance from 0

4
0

Both -4 and 4 are 4 units from 0, so | -4 | = 4 and | 4 | = 4

Key Properties of Absolute Value

  • |x| ≥ 0 for all x (absolute value is always non-negative)
  • |x| = 0 if and only if x = 0
  • |xy| = |x||y|
  • |x/y| = |x|/|y| for y ≠ 0
  • |x + y| ≤ |x| + |y| (Triangle Inequality)

Solving Basic Absolute Value Equations

For equations of the form |x| = a, where a is a positive real number:

|x| = a ⇔ x = a or x = -a

Example 1: |x| = 5

|x| = 5
x = 5 or x = -5

Solution: x = 5 or x = -5

-5
5

Example 2: |2x - 3| = 7

|2x - 3| = 7
Case 1: 2x - 3 = 7
2x = 10 → x = 5
Case 2: 2x - 3 = -7
2x = -4 → x = -2

Solution: x = 5 or x = -2

Important Note

If |expression| = a negative number, there is no solution because absolute value is always non-negative.

Example: |x| = -5 has no solution.

Practice Problem

|3x + 2| = 8
Case 1: 3x + 2 = 8
3x = 6 → x = 2
Case 2: 3x + 2 = -8
3x = -10 → x = -10/3

Solution: x = 2 or x = -10/3

Solving Basic Absolute Value Inequalities

For inequalities with absolute values, we consider two cases based on the properties of absolute value:

  • |x| < a ⇔ -a < x < a (for a > 0)
  • |x| > a ⇔ x < -a or x > a (for a > 0)

Example 1: |x| < 4

|x| < 4
-4 < x < 4

Solution: All numbers between -4 and 4

-4
4

Example 2: |x| ≥ 3

|x| ≥ 3
x ≤ -3 or x ≥ 3

Solution: All numbers less than or equal to -3 or greater than or equal to 3

-3
3

Practice Problem

Solve: |2x - 1| ≤ 5

|2x - 1| ≤ 5
-5 ≤ 2x - 1 ≤ 5
-5 + 1 ≤ 2x ≤ 5 + 1
-4 ≤ 2x ≤ 6
-2 ≤ x ≤ 3

Solution: All numbers between -2 and 3, inclusive

Advanced Absolute Value Equations

More complex absolute value equations may require:

  • Isolating the absolute value expression first
  • Considering special cases when the expression equals 0
  • Checking for extraneous solutions

Example 1: |2x - 4| + 3 = 9

|2x - 4| + 3 = 9
|2x - 4| = 6 (Isolate the absolute value)
Case 1: 2x - 4 = 6
2x = 10 → x = 5
Case 2: 2x - 4 = -6
2x = -2 → x = -1

Solution: x = 5 or x = -1

Example 2: |3x + 2| = |x - 1|

|3x + 2| = |x - 1|
Case 1: 3x + 2 = x - 1
2x = -3 → x = -3/2
Case 2: 3x + 2 = -(x - 1)
3x + 2 = -x + 1
4x = -1 → x = -1/4

Solution: x = -3/2 or x = -1/4

Special Case: |expression| = 0

If |expression| = 0, then the expression must equal 0.

Example: |2x - 6| = 0 → 2x - 6 = 0 → x = 3

Practice Problem

Solve: 2|4x - 3| - 5 = 7

2|4x - 3| - 5 = 7
2|4x - 3| = 12
|4x - 3| = 6
Case 1: 4x - 3 = 6
4x = 9 → x = 9/4
Case 2: 4x - 3 = -6
4x = -3 → x = -3/4

Solution: x = 9/4 or x = -3/4

Advanced Absolute Value Inequalities

More complex absolute value inequalities may require:

  • Isolating the absolute value expression first
  • Considering the direction of the inequality
  • Solving compound inequalities
  • Checking solutions in the original inequality

Example 1: |2x - 3| > 7

|2x - 3| > 7
Case 1: 2x - 3 > 7
2x > 10 → x > 5
Case 2: 2x - 3 < -7
2x < -4 → x < -2

Solution: x < -2 or x > 5

-2
5

Example 2: |3x + 1| + 2 ≤ 8

|3x + 1| + 2 ≤ 8
|3x + 1| ≤ 6
-6 ≤ 3x + 1 ≤ 6
-7 ≤ 3x ≤ 5
-7/3 ≤ x ≤ 5/3

Solution: All numbers between -7/3 and 5/3, inclusive

-7/3
5/3

Special Case: |expression| < negative number

If |expression| < a negative number, there is no solution.

Example: |x| < -3 has no solution.

Practice Problem

Solve: |2 - 3x| ≥ 4

|2 - 3x| ≥ 4
Case 1: 2 - 3x ≥ 4
-3x ≥ 2 → x ≤ -2/3
Case 2: 2 - 3x ≤ -4
-3x ≤ -6 → x ≥ 2

Solution: x ≤ -2/3 or x ≥ 2
